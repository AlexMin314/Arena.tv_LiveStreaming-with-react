# Mind Tap with react
GA WDI Final Project - A Pictionary-like, fun and engaging game


### Introduction
* Pictionary-like real-time drawing/guessing multiplayer game. Draw and Guess the keyword with your creativity!

### Stacks
* Canvas
* React
* Redux
* Firebase

### Getting Start
* How to install
  - npm install
  - npm start
